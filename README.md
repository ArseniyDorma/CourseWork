# Датасет #1
**Название:** Работа с базой данных (Database & NLP).  
**Тип:** Табличные данные.  
**Требование:** Создать систему, позволяющую пользователю задавать вопросы на естественном языке и получать соответствующие данные из базы данных.  
**Исходные данные:** Вопрос от пользователя, SQL-запрос для получения ответа, контекст запроса.

# Датасет #2
**Название:** Классификация спама по тексту электронного письма.  
**Тип:** Текст.  
**Требование:** Построить модель классификации, определяющую, является ли письмо спамом.  
**Исходные данные:** Тексты электронных писем, преобразованные в вектор признаков на основе словаря из 5 000 слов. Разметка: спам / не спам.

# Датасет #3
**Название:** Диагностика подшипников по вибрационным данным.  
**Тип:** Временные ряды.  
**Требование:** Построить модель, использующую пространственно-временные паттерны вибрационных сигналов. Модель должна локализовать источник аномалии и работать с выбросами и шумом.  
**Исходные данные:** Временные ряды вибрации с 25 датчиков (акселерометров). Классы: норма, дефект внутреннего кольца подшипника, дефект тела качения подшипника, дефект наружного кольца подшипника.  

# Датасет #4
**Название:** Разработать модель, способную идентифицировать средства индивидуальной защиты (СИЗ).  
**Тип:** Изображение.  
**Требование:** Построить модель компьютерного зрения, определяющую, присутствуют ли на изображении средства индивидуальной защиты (каска, перчатки, жилет, очки и т.д.) и соответствуют ли они нормам безопасности.  
**Исходные данные:** Набор изображений с производственных объектов, строительных площадок или складов, размеченных с указанием классов объектов (тип СИЗ) и их локализации (bounding boxes).  

# Датасет #5
**Название:** Создание инструментов ИИ для диагностики развития речи (коммуникативное развитие).  
**Тип:** Аудио.  
**Требование:** Построить модель, способную оценивать уровень коммуникативного развития ребёнка на основе речевых образцов, выявлять задержки речевого развития и классифицировать нарушения.  
**Исходные данные:** Аудиозаписи спонтанной и направленной речи детей разных возрастных групп, транскрипции высказываний, аннотированные по параметрам: словарный запас, грамматическая структура, фонетическая правильность, длина фразы, наличие специфических речевых ошибок. Разметка: уровень речевого развития (норма / задержка / нарушение) или интегральная оценка по шкале коммуникативного развития.
